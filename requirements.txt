# SETUP INSTRUCTIONS (fresh computer):
#   1. Python 3.10 or 3.11 recommended  (3.12+ may have numba/llvmlite issues)
#   2. Create venv:   python -m venv venv
#   3. Activate:      venv\Scripts\activate        (Windows)
#                     source venv/bin/activate      (Linux/Mac)
#   4. Install deps:  pip install -r requirements.txt
#   5. Install FFmpeg (external, required by pydub):
#        Windows: winget install ffmpeg   OR  download from https://ffmpeg.org
#        Linux:   sudo apt install ffmpeg
#        Mac:     brew install ffmpeg
#   6. Run server:    python app.py
#      → On FIRST RUN the PANNS CNN14 model (~322 MB) downloads automatically
#        to ~/panns_data/  — internet required for this one-time download.
# ──────────────────────────────────────────────────────────────────────────────
